## 受講者へ

各講義フォルダの training フォルダに演習問題があります。演習問題を解いたら（解いている途中でもいいので）、各問題のフォルダに、自分の名前を付けたファイル (例:yingtai.hs) を提出してください。

kinokkory.hs が僕の解答例です。行き詰ったらちらっと参考にしてくれても構いません。kinokkory.mdは問題の解説です。
